# MCDatapackGenerator
Python script to generate custom items for Minecraft in the form of a datapack

## Use
Currently this only works with the Minecraft Jar being extracted into this project's folder, with it's name matching the vanilla_folder variable.

Folder paths may not be always generated, and may crash if they aren't manually made.

Put all custom textures into the folder named "`assets`"
